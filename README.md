# AWS Terraform Infrastructure Project

This project demonstrates how to deploy AWS infrastructure using Terraform.

## Project Overview
This project provisions AWS resources using Infrastructure as Code (IaC).

## Technologies Used
- AWS
- Terraform
- AWS Lambda
- Amazon S3
- IAM
- CloudWatch

## Project Structure
main.tf – Main infrastructure configuration  
variables.tf – Terraform variables  
outputs.tf – Terraform outputs  

## Deployment Steps

1. Install Terraform
2. Configure AWS CLI
3. Run:

terraform init

terraform plan

terraform apply
